# markdown 语法的基本应用

一级标题的运用 ：# 后加上标题的文字

二级标题的运用：## 后加上标题的文字

其他标题的运用以此类推，最多为六级标题

# 我是二级标题

# 我是三级标题

# 四级标题

设置字体的格式为斜体：在文字的左右两边加上 * 或者 在文字的左右两边加上 _

​ hello
​ hello

删除线的运用：在文字的左右加上英文状态下的～～

​ 删除

给文字高亮的运用：在文字的左右两边都加上等号 =

​ 高亮

设置文字格式为粗体： 在文字的左右两边均加上 **

​ 我是加粗体

链接的使用方法:[链接名称] 加上（链接地址）

​ 百度

表情符号的使用：一个冒号：后表情的英文，如:smile 、:cry

​ 😏😄😢

引用的格式 一个英文状态下的右尖尖符号 >

无序列表的运用：* 加上空格

  • 无序列表 1
  • 无序列表 2

有序列表的运用：数字加上英文状态下的句号。再加上空格（一定要加上空格，否则不生效）

  1. 有序列表 1
  2. 有序列表 2

代码块的运用： 英文状态下的 ``` 加上代码的语言

分割线的运用 1：*** 后回车

分割线的运用 2：___后回车

分割线的运用 3：--- 后回车
